# Docker
### Build
```bash
cd ~
git clone https://github.com/wisehackermonkey/ascii-pipe.git
cd ascii-pipe
docker build -t wisehackermonkey/ascii-pipe:latest .  
```
### Run
```bash
docker run -it --rm --name wisehackermonkey/ascii-pipe:latest  
```
### Docker-compose
```bash
docker-compose build
docker-compose up
```
